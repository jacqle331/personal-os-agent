# Setup Checklist

- [ ] GitHub account created
- [ ] GitHub email verified + 2FA enabled
- [ ] An AI assistant available (Copilot, Claude Code, or Gemini CLI)
- [ ] Node.js installed (`node --version` works)
- [ ] Assistant CLI installed
- [ ] Assistant command works in terminal (`copilot` / `claude` / `gemini`)
- [ ] Git installed and `git --version` works
- [ ] PowerShell 7+ or compatible shell available
- [ ] Repo created from template
- [ ] Working files generated (`scripts/setup.ps1` or `setup.sh`)
- [ ] Placeholders replaced in intake/template files
- [ ] Instructions file present for your assistant
- [ ] Launch aliases configured (optional — life/finance/relationship)
- [ ] First check-in completed
- [ ] Backup path configured (OneDrive/Drive/etc.)
