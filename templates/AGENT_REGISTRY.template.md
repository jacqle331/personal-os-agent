## How to invoke
Each agent is a *mode* of the same Copilot setup. Start a session and say, e.g.:
- "Life mode — plan my week"
- "Finance mode — bill check-in"
- "Relationship mode — help me prep a conversation"

See `docs/ALIASES.md` to set up one-word launch shortcuts.
