# Map Your Recurring Tasks

The fastest way to get value from your agents is to hand them the things you do
**over and over**. Spend 10 minutes listing your routine, then let the right
agent support each item.

## How to do this
1. List the tasks you handle daily, weekly, and monthly (brain-dump, no editing).
2. For each, pick the agent that fits: **Life**, **Finance**, or **Relationship**.
3. Add the recurring ones to your cadence in `AGENT_REGISTRY.md` and note the
   active ones in `CURRENT_CONTEXT.md`.
4. Or just paste your list into a session and say: *"Map these to my agents and
   suggest a cadence."*

## Common tasks → which agent helps

### 🟢 Life Agent
| You regularly… | How the agent supports it |
|----------------|---------------------------|
| Plan the week ahead | Builds your top 3 priorities from current context |
| Juggle appointments, school, activities | Tracks them and surfaces what's next |
| Run errands / chores | Turns them into a simple ordered list |
| Tackle a big project | Breaks it into small next actions |
| Lose track of follow-ups | Carries open tasks forward each week |

### 🔵 Finance Agent (rhythm, not advice)
| You regularly… | How the agent supports it |
|----------------|---------------------------|
| Pay bills | Walks a checkpoint: what's due, what cleared |
| Watch the budget | Compares categories to what you actually spent |
| Overspend in spots | Flags it and suggests small adjustments |
| Save toward a goal | Tracks progress and keeps it visible |
| Prep for a money talk | Summarizes the month in plain language |

### 🟠 Relationship Agent
| You regularly… | How the agent supports it |
|----------------|---------------------------|
| Navigate hard conversations | Helps you phrase things calmly and clearly |
| Cool down before reacting | Thinks it through with you first |
| Plan time together | Suggests and schedules a check-in or date |
| Reflect on the week | Prompts a short, honest review |

## Make it a habit
Once mapped, the recurring items become your **check-in agenda**. Each session,
your agent reads `CURRENT_CONTEXT.md` and `SESSION_LOG.md` so nothing falls
through the cracks. Update the cadence anytime your life changes.
